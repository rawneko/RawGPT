**RawGPT** — это современный интерфейс для локальных нейросетей (Ollama). Полная приватность, стильный дизайн "Opera Air" и мощный функционал. Всё работает локально на вашем ПК.

![Скриншот интерфейса](./screenshot.png)

(ненавижу улучшение сайта по улучшению качества фото через нейронку)

## ✨ Особенности

* Современный интерфейс с Glassmorphism, блюром и плавными анимациями.
* **Полная приватность** — История чатов и настройки хранятся только локально.
* **Ollama Интеграция** — Автоматическое подключение, список моделей, загрузка функций (Vision, Tools).
* Поддержка загрузки изображений и файлов (Drag & Drop).
* **Markdown & Code** — Красивая подсветка синтаксиса кода с кнопкой копирования.
* **Темы оформления** — Dark, Light и AMOLED (True Black).
* Оптимизированный рендеринг, потоковый вывод текста.

## 🛠 Установка

### Требования
1.  Установленная **[Ollama](https://ollama.com/)**.
2.  Windows 10 или 11.

### Скачать
Перейдите в раздел **Releases** и скачайте последнюю версию `RawGPT.exe`.

[Download the RawGPT for android](https://github.com/rawneko/RawGPT-android/releases)

### Запуск из исходного кода

```bash
# 1. Клонировать репозиторий
git clone https://github.com/rawneko/RawGPT.git

# 2. Перейти в папку
cd RawGPT

# 3. Установить зависимости
npm install

# 4. Запустить в режиме разработки
npm start

# 5. Собрать .exe файл
npm run build
```
